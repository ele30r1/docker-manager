# A simple docker mamangement tool

This is a web UI for docker, focused on conatiners.

## Features

- List images on docker host
- Search in dockerhub and pull an image
- List containers on docker host
- Show details of each container including status, memory usage, ...
- Controlling the state of each container
- Committing a continer

## Deployment

Simply use docker compose to start services:

```bash
docker compose up
```
